Servicios de Escritorio movistar

Descripción detallada

«« AnteriorSiguiente »»


Los servicios disponibles en Escritorio movistar puedes verlos en la barra de servicios, situada en la zona central de la ventana de la aplicación.

Pulsando con el botón secundario del ratón sobre alguno de los servicios verás un menú contextual que te permite Abrir, establecer la Configuración, Instalar u obtener Más información relativa al servicio servicio correspondiente.

De forma predeterminada, cuando un servicio no se encuentre instalado no aparecerá en la barra de servicios. Esta opción puedes modificarla desde Configuración >> Servicios, activando/desactivando la casilla Ocultar servicios no instalados.

A continuación te describimos cada uno de ellos de forma detallada.

 

movistar Internet

Este servicio te proporciona acceso general a Internet. En Configuración >> movistar Internet tienes la opción de elegir si abrir o no el navegador predeterminado de tu ordenador tras establecer la conexión mostrando el sitio web que tengas configurado como Página de inicio (de forma predeterminada, http://www.movistar.es).

 

Mensajes

Puedes enviar y recibir mensajes SMS desde tu ordenador mediante el servicio de Mensajes, tanto si te conectas a través de una Tarjeta o un Módem USB Internet Móvil. Si te conectas a través de un Teléfono sólo podrás enviar mensajes.

Consulta aquí cómo funciona el servicio de Mensajes.

 

Correo Móvil

El servicio de Correo Móvil establece una conexión y abre el servicio de Correo Móvil usando la interfaz web o, de forma opcional, tu cliente de correo predeterminado, según lo seleccionado en el apartado Configuración >> Correo Móvil que puedes configurar en Configuración >> Datos de Usuario. Este servicio precisa de tus datos de autenticación (teléfono y clave).

 

Zonas Wi-Fi de Telefónica

Con cualquiera de las Tarifas Banda Ancha Móvil de calidad Premium podrás conectarte en las Zonas Wi-Fi de Telefónica gratuitamente y de forma ilimitada.

Las Tarifas con las que podrás disfrutar del acceso Wi-Fi son las siguientes:

  • Internet en el Móvil Plus
  • Tarifa Plana Internet Plus
  • Tarifa Plana Internet Premium

Pulsando el botón Cobertura podrás seleccionar tu ubicación y así acceder a la lista de Zonas Wi-Fi de Telefónica (listado actualizado a mayo de 2008). Si dispones de un acceso a Internet, puedes consultar el listado actualizado en la dirección URL http://www.telefonicaonline.com/zonawifi

Si has contratado alguna de las tarifas anteriores, pulsando el botón Solicitar clave obtendrás una clave para conectarte.

 

movistar Intranet

Este servicio está concebido para realizar conexiones a la intranet corporativa de tu empresa. De forma predeterminada, este servicio no está configurado. Si quieres configurarlo puedes hacerlo desde Configuración >> movistar Intranet, para lo cual deberás ponerte en contacto con el administrador de la red de tu empresa para que te facilite los datos de conexión.

 

Si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que ésa sea tu clave de acceso.
