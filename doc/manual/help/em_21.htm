Instalar el software de Escritorio movistar

Requisitos de instalación

«« AnteriorSiguiente »»


Las características mínimas con las que debe contar tu ordenador para ser compatible con el servicio Escritorio movistar 7.0 para Linux son las siguientes:

Hardware:

  • Ordenador PC o compatible con ranura PCMCIA o ExpressCard para Tarjeta Internet Móvil o puerto USB 2.0 para Módem USB Internet Móvil o puerto de conexión al Teléfono que deseas utilizar (Bluetooth, USB, puerto serie o infrarrojos).
  • Tarjeta SIM dada de alta en el servicio GPRS/3G de movistar. 
  • Dispositivo Internet Móvil, ya sea una Tarjeta o Módem USB Internet Móvil o un Teléfono con conexión GPRS/3G.

Sistema Operativo:

  • Linux,distribuciones soportadas: Fedora Core 10 (con versión de Kernel  2.6.27.5-117),  Ubuntu 8.10 (con versión de Kernel 2.6.27-9) y OpenSUSE 11.1 (con versión de Kernel 2.6.27).

 

Comprueba que tu Dispositivo USB Internet Móvil dispone de un firmware actualizado. Si no estuviera actualizado, podrás descargar las últimas versiones desde la dirección URL: http://www.movistar.es

Para utilizar el servicio Escritorio movistar debes ser cliente de Telefónica Móviles España y estar dado de  alta en el servicio GPRS/3G. Para comprobar que estás dado de alta en el servicio, puedes llamar al número de atención al cliente (609) de movistar.
